		Documento de Arquitetura de Software

1. Introdução
   Sobre o projeto Cartório EBAC, tem como objetivo ser organizador
   para o cliente, facilitando o gerenciamento dos dados cadastrados.
   O programa possui três requisitos: função de registro de novos usuários,
   coletando irformações como nome, cpf e profissão; consulta de nomes
   cadastrados com base em cpf; e opção de deletar usuários, também com base
   em cpf. A linguagem escolhida para o projeto foi o C.
